Privacy Policy

GitHub HTML Preview Local  ·  Last updated: May 2025

Summary

GitHub HTML Preview Local does not collect, store, or transmit any personal data or user information.

Data Collection

This extension does not collect any data of any kind. Specifically:

  • No personal information is collected.
  • No browsing history is recorded.
  • No usage data or analytics are gathered.
  • No data is sent to any external server.

How the Extension Works

The extension reads the HTML source code already displayed in your browser on GitHub file pages and renders it locally within the same page. All processing happens entirely within your browser. No content from your browser is transmitted outside of your device.

Permissions

The extension requests the following permissions solely to provide its core functionality:

  • activeTab — to read the HTML source displayed on the current GitHub page.
  • scripting — to inject the Preview button and render the preview inline.
  • https://github.com/* — to operate on GitHub file view pages.

These permissions are not used to collect or transmit any data.

Third Parties

This extension does not share any data with third parties because no data is collected.

Contact

If you have questions about this privacy policy, please open an issue on the GitHub repository.
